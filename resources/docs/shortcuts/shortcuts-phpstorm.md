## Window Manager Shortcuts

Command | Shortcut
--------|---------
Go to class | `Ctrl + N`
Got to file | `Ctrl + Shift + N`
Go to symbol | `Ctrl + Shift + Alt + N`
Go to line | `Ctrl + G`
Go to actions | `Ctrl + Shift + A`
Show Sidebar | `Akt + 1`
Show Sidebar Git | `Alt + 9`
Show Terminal | `Alt + F12`
Close Editor | `Ctrl + F4`
