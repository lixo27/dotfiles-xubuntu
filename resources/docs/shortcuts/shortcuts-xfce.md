## Keyboard Shortcuts

Command | Shortcut
--------|---------
Text Editor | `Super + E`

Terminal Emulator | `Super + T`
Web Browser | `Super + W`
Whisker Menu | `Super + Space`
Lock Screen | `Super + L`
Code Editor | `Super + C`
PhpStorm IDE | `Super + S`

Command | Shortcut

Upper Workspace | `Ctrl + Alt + Up`
Bottom Workspace | `Ctrl + Alt + Down`
Left Workspace | `Ctrl + Alt + Left`
